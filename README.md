# TpIntegrador-PAW2026

**Materia:** PAW
**Fecha de Entrega:** 27/03/2026  

## Introducción

Esta primera entrega tiene como objetivo dar inicio al desarrollo de una Aplicación Web orientada a la adopción de perros. 
En este repositorio se puede encontrar la propuesta general de la aplicación a desarrollar en forma de presupuesto funcional y temporal,
y el planteo de un **sitemap inicial**, que servirá como guía para estructurar 
las secciones del sitio y planificar la navegación del usuario.
