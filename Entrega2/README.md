**Materia:** PAW

## Introducción

Esta segunda entrega tiene como objetivo la presentación de los *Wireframes principales* de la aplicación **PawMap**, además de la elección de la **arquitectura de aplicación** y el diseño del **modelo de objetos**.
En este repositorio se pueden encontrar los **wireframes Pawmap**, la **definición de la arquitectura de la aplicación** y el **diseño del modelo de objetos del sistema**. Estos elementos sirven como base para la futura implementación del sistema y para estructurar el desarrollo de las funcionalidades planificadas.
