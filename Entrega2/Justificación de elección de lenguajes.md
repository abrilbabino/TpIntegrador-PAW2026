# Justificación de elección de lenguajes  
## Elección de lenguaje para el Backend: Java  
Elegimos Java porque:   
- Tiene compatibilidad directa con JDBC para el acceso a la base de datos.  
- Nos permite separar fácilmente el Tier dos en una arquitectura MVC.  
- Tiene una alta estabilidad para aplicaciones web.  
- Tiene un tipado fuerte que reduce errores en las aplicaciones.

## Elección de lenguaje para la Base de Datos: PostgreSQL  
Elegimos PostgreSQL porque:
- Es una base de datos relacional robusta (ACID) con integridad referencial y soporte completo de SQL.
- Se integra correctamente con JDBC.
- Es de Software libre (No requiere licencias).
- Tiene alto rendimeinto y escalabilidad.
- Tiene un amplio soporte de tipos de datos: JSON, arrays, entre otros.
